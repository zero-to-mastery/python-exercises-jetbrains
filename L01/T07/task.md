# List Indexing and Modification

Lists store multiple values in a single variable. You can access and modify list elements using their index position.

**Accessing elements:**

```python
fruits = ["apple", "banana", "cherry"]
fruits[0]     # 'apple' - first element
fruits[-1]    # 'cherry' - last element
```

**Slicing lists:**

```python
numbers = [1, 2, 3, 4, 5]
numbers[1:3]    # [2, 3] - elements from index 1 to 2
```

**Modifying elements:**

```python
colors = ["red", "blue", "green"]
colors[1] = "yellow"    # colors is now ["red", "yellow", "green"]
```

**Concatenating lists:**

```python
list1 = [1, 2]
list2 = [3, 4]
combined = list1 + list2    # [1, 2, 3, 4]
```
