# List Methods

Lists in Python have built-in methods that allow you to modify and query their contents. These methods operate directly on the list and change it in place.

## Common List Methods

- `list.remove(item)` - Removes the first occurrence of the specified item from the list
- `list.append(item)` - Adds an item to the end of the list
- `list.insert(index, item)` - Inserts an item at the specified index position
- `list.count(item)` - Returns the number of times an item appears in the list
