# Your First Python Program

Welcome! This is your first Python program. Let's learn how input and output work.

## What This Program Does

This program will:
1. Ask you for your name
2. Greet you with a personalized hello message

## How to Run Your Program

**Step 1: Find the Run Button**
- Look for the **green triangle** (▶) button at the top right of PyCharm
- Or right-click on the file and select "Run"

**Step 2: Run the Program**
- Click the green triangle button
- The **Run tool window** will open at the bottom of your screen

**Step 3: Interact with Your Program**
- You'll see the question: "what is your name?"
- **Type your name** in the Run tool window (the area at the bottom)
- Press **Enter**

**Step 4: See the Result**
- Your program will print: "Hello [your name]"

## Understanding the Code

```python
name = input("what is your name?\n")  # Gets input from you
print("Hello " + name)                 # Prints the greeting
```

- `input()` - Asks you to type something
- The Run tool window (at bottom) is where you type and see output
- `print()` - Shows messages on screen

## Try It!

1. Click the green triangle ▶ at the top right
2. Type your name when asked
3. Press Enter
4. See your personalized greeting!

**Tip:** You can run the program as many times as you want. Try different names!
