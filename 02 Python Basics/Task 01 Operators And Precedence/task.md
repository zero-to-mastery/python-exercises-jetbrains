## Operator Precedence
Python evaluates operators in this order (highest to lowest):
1. `**` (exponentiation)
2. `*`, `/`, `//`, `%` (multiplication, division, floor division, modulo)
3. `+`, `-` (addition, subtraction)
4. Parentheses `()` override the default order
