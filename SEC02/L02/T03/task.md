# String Indexes

## What You'll Learn

Access individual characters in strings using index positions.

## Concept Overview

Strings are sequences of characters that can be accessed by their position using indexing and slicing.

## Key Concepts

### Zero-Based Indexing
Python uses zero-based indexing where the first character is at index 0.

## Example

```python
word = "Python"
first = word[0]  # 'P'
```
