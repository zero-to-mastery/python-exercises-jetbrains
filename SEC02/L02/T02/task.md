# Formatted Strings

## What You'll Learn

Create dynamic strings using formatting techniques.

## Concept Overview

String formatting allows you to create strings with variable content, making your code more dynamic and readable.

## Key Concepts

### F-Strings
Modern Python uses f-strings for clean, readable string formatting.

## Example

```python
name = "Alice"
greeting = f"Hello, {name}!"
```
