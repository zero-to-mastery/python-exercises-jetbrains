# List Slicing

## What You'll Learn

Extract portions of lists using slicing notation.

## Concept Overview

List slicing allows you to extract sublists using start:stop:step notation.

## Key Concepts

### Slice Notation
Use [start:stop:step] to extract elements from lists.

## Example

```python
numbers = [0, 1, 2, 3, 4, 5]
first_three = numbers[0:3]
```
