# Operator Precedence

## What You'll Learn

Understand how Python evaluates expressions with multiple operators using precedence rules.

## Concept Overview

Operator precedence determines the order in which operations are performed in expressions. This is crucial for writing correct mathematical expressions.

## Key Concepts

### Precedence Rules
Python follows standard mathematical precedence: parentheses, exponents, multiplication/division, addition/subtraction (PEMDAS).

## Example

```python
result = 2 + 3 * 4  # Result is 14, not 20
```
