# Type Conversion

## What You'll Learn

Convert between different data types in Python.

## Concept Overview

Type conversion (casting) allows you to convert values from one type to another, essential for working with mixed data types.

## Key Concepts

### Built-in Conversion Functions
Use int(), float(), str() to convert between types.

## Example

```python
age_str = "25"
age_int = int(age_str)
```
