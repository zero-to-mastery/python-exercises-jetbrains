# Integers and Floats

## What You'll Learn

Work with numeric data types in Python, understanding integers and floating-point numbers.

## Concept Overview

Python supports two primary numeric types: integers (whole numbers) and floats (decimal numbers). Understanding these types is fundamental to performing calculations and working with numerical data.

## Key Concepts

### Integer Type
Integers represent whole numbers without decimal points. They can be positive, negative, or zero.

### Float Type
Floats represent real numbers with decimal points, essential for precise calculations.

## Example

```python
age = 25  # integer
price = 19.99  # float
```
