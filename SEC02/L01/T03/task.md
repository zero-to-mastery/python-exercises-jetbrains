# Augmented Assignment Operator

## What You'll Learn

Learn shorthand operators that combine arithmetic operations with assignment.

## Concept Overview

Augmented assignment operators provide a concise way to update variables by combining an operation with assignment.

## Key Concepts

### Common Augmented Operators
Operators like +=, -=, *=, /= modify a variable in place.

## Example

```python
count = 5
count += 3  # Same as: count = count + 3
```
