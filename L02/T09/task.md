# Introduction to Lists

## What is a list?
A list is an **ordered** and **mutable** collection of items in Python.  
Lists use square brackets `[]`.

## How to create a list
Write square brackets and put elements inside, separated by commas:
- Empty list: `[]`
- List with elements: `[item1, item2, item3]`

## Indexing starts at 0
List positions (indexes) start from 0, not 1:

`list1[0]` is the first element  
`list1[1]` is the second element  
`list1[-1]` is the last element (negative indexes count from the end)

## Lists can store different types
A list can contain items of different data types (numbers, strings, booleans, etc.).
