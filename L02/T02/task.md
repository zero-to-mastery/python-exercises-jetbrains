## Augmented Assignment Operators

Shorthand for updating a variable:
- `x += 5` is the same as `x = x + 5`
- `x -= 3` is the same as `x = x - 3`
- `x *= 2` is the same as `x = x * 2`
- `x /= 4` is the same as `x = x / 4`
